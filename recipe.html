How to make a page for the Web

Ingredients

  • Blank HTML file
  • Colors
  • Typeface
  • Layout
  • Animation
  • Interaction

Instructions

  1. Open your blank HTML file.
  2. Add text, images, and other content.
  3. Mark up your content to show relationships.
  4. Use a style sheet to change colors, margins and the typeface.
  5. (Optional) Add animation and interaction to make the page more lively.
